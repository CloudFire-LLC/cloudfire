# Windows Client

Firezone supports Microsoft Windows with a native client for the Intel/AMD x86_64
architecture.

## Prerequisites

- Windows 10 or higher
- x86_64 CPU

## Installation

The Windows client is currently in beta and can be downloaded from our
[main repository's releases page](https://github.com/firezone/firezone/releases).

Use the MSI to ensure that WebView2 is installed. Installing the exe manually
is not recommended.
